На восьмом занятии мы научились позиционированию и построению CSS-таблиц

Таблица свойств CSS
НазваниеПримерОписание свойства
display.div { display: table; }Многоцелевое свойство, которое определяет, как элемент должен быть показан в документе. C помощью значения "table" определяет, что элемент является блочной таблицей. с помощью значения "table-cell" указывает, что элемент представляет собой ячейку таблицы
float.layer { float: left; }Определяет, по какой стороне будет выравниваться элемент, при этом остальные элементы будут обтекать его с других сторон. Когда значение свойства float равно none, элемент выводится на странице как обычно, при этом допускается, что одна строка обтекающего текста может быть на той же линии, что и сам элемент. Основные значения - left | right | none | inherit
cleardiv style="clear: left"Устанавливает, с какой стороны элемента запрещено его обтекание другими элементами. Если задано обтекание элемента с помощью свойства float, то clear отменяет его действие для указанных сторон. Основные значения - left | right | both | inherit
position.example { position: relative; }Устанавливает способ позиционирования элемента относительно окна браузера или других объектов на веб-странице. Основные значения - absolute (Положение задается left, top, right и bottom, элемент теряет свое место в блоке) | fixed (Привязывается к заданой позиции и не меняет положения во время прокрутким страницы) | relative (Положение элемента устанавливается относительно его исходного места. Своё место элемент не теряет) | static (Элементы отображаются как обычно) | inherit
z-index#layer1 { z-index: 2; }Любые позиционированные элементы на веб-странице могут накладываться друг на друга в определенном порядке, имитируя тем самым третье измерение, перпендикулярное экрану. Каждый элемент может находиться как ниже, так и выше других объектов веб-страницы, их размещением по z-оси и управляет z-index. Это свойство работает только для элементов, у которых значение position задано как absolute, fixed или relative.
