Таблица свойств CSS
НазваниеПримерОписание свойства
font-sizep { font-size: 11pt; }Определяет размер шрифта элемента
colorp { color: rgb(49, 151, 116); }Определяет цвет текста элемента
font-weighth1 { font-weight: normal; }Устанавливает насыщенность шрифта. Значение устанавливается от 100 до 900 с шагом 100
font-stylep { font-style: italic; }Определяет начертание шрифта — обычное, курсивное или наклонное
font-familyh1 { font-family: Geneva, Arial, Helvetica, sans-serif; } Устанавливает семейство шрифта, которое будет использоваться для оформления текста содержимого
text-decorationa { text-decoration: underline; }Добавляет оформление текста в виде его подчеркивания, перечеркивания, линии над текстом и мигания
text-indentP { text-indent: 1.5em; }Устанавливает величину отступа первой строки блока текста
line-heighth1 { line-height: 60%; }Устанавливает интерлиньяж (межстрочный интервал) текста, отсчет ведется от базовой линии шрифта
letter-spacingp { letter-spacing: 5px; }Определяет интервал между символами в пределах элемента
word-spacingp { word-spacing: 20px; }Устанавливает интервал между словами

background-sizediv { background-size: cover; }Масштабирует фоновое изображение согласно заданным размерам
background-positionbody { background-position: right bottom; }Задает начальное положение фонового изображения, установленного с помощью свойства background-image
background-repeatbody { background-repeat: repeat-x; }Определяет, как будет повторяться фоновое изображение, установленное с помощью свойства background-image
background-attachmentbody background-attachment: fixed; }Свойство background-attachment устанавливает, будет ли прокручиваться фоновое изображение вместе с содержимым элемента
