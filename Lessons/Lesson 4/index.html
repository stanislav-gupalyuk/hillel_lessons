На четвёртом занятии мы изучали построение блочных конструкций и изменение их визуальных свойств с помощью CSS

Таблица свойств CSS
НазваниеПримерОписание свойства
padding.layer { padding: 20px; }Устанавливает значение полей вокруг содержимого элемента. Полем называется расстояние от внутреннего края рамки элемента до воображаемого прямоугольника, ограничивающего его содержимое
marginbody { margin: 0; }Устанавливает величину отступа от каждого края элемента. Отступом является пространство от границы текущего элемента до внутренней границы его родительского элемента
display.exampleTitle { display: inline; }Многоцелевое свойство, которое определяет, как элемент должен быть показан в документе
vertical-align.logo { vertical-align: middle; }Выравнивает элемент по вертикали относительно своего родителя, окружающего текста или ячейки таблицы
